## Java AI Sandbox

---
Sandbox for AI application

### Version 0.0.3

#### Features
- Claude Thinking Stream UI at `/chat`
- Real-time streaming of AI thinking process and answers
- Light/dark theme toggle (persisted in localStorage)
- [Embabel](https://github.com/embabel/embabel-agent) agent framework integration
- Fitness planning agent (`POST /api/fitness/plan`) that turns a free-text goal
  description into a personalized workout + nutrition plan via goal-oriented
  action planning
